# TASK QUEUE

締切・重要度・重さ（作業量）から優先度を自動計算するタスク管理PWA。
「今やるべきタスク」を自動で提示します。

## 特徴

- **タスク / やりたいこと / 日課** の3モード
- スマート順ソート（締切 × 重要度 × 重さ）
- ダーク / ライトテーマ
- JSONエクスポート / インポート
- PWA対応（オフライン動作・ホーム画面に追加でアプリ化）
- データは端末内にローカル保存（サーバー送信なし）

## 使い方

ブラウザでアプリURLを開き、ホーム画面に追加するだけ。

- **iPhone**: 共有ボタン → 「ホーム画面に追加」
- **Android**: メニュー → 「ホーム画面に追加」

## ローカル開発

```bash
npm install
npm run dev
```
